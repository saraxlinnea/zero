# Zero

This is the official specimen record for Zero, the goodest little boy in the world. He is a Samoyed in San Francisco who goes by many names, loves head massages, and has strong opinions about harnesses.

**Live site:** https://saraxlinnea.github.io/zero/

## What's on the site

- Profile, tricks, breed facts, photo gallery, and tick tracker
- Cutouts of Zero in various dignified poses
- A record of his biggest adventures

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173/zero/

## Add photos

Drop new images in `Photos/`, then run:

```bash
npm run photos
```

## Deploy

Pushes to `main` deploy automatically to GitHub Pages.
